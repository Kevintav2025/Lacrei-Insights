# 💜 Lacrei Insights - Dashboard de Impacto Social
> 🔗 **Acesse o projeto online:** [https://kevintav2025.github.io/Lacrei-Insights/]

O **Lacrei Insights** é um dashboard interativo desenvolvido para monitorar atendimentos e profissionais da plataforma Lacrei. Ele oferece uma visão clara e dinâmica sobre o crescimento das consultas, distribuição de especialidades e gestão de prontuários.

## 🚀 Funcionalidades

- **Gráficos Dinâmicos**: Visualização de crescimento de consultas (Alternância entre 6 e 12 meses) usando Chart.js.
- **Gestão de Profissionais**: Tabela interativa com status (Online/Offline) e busca em tempo real por nome ou especialidade.
- **Sistema de Prontuários**: Modal interativo para visualização de detalhes específicos de cada atendimento.
- **Distribuição de Especialidades**: Gráfico de rosca (Doughnut) para análise de impacto por área médica.
- **Barra de Progresso**: Monitoramento de metas mensais de atendimento.
- **Design Responsivo**: Adaptado para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

* **HTML5** - Estruturação semântica.
* **CSS3** - Estilização moderna com variáveis e Flexbox/Grid.
* **JavaScript (ES6+)** - Lógica de filtros, manipulação de DOM e interatividade.
* **Chart.js** - Biblioteca para renderização de gráficos.
* **Lucide Icons** - Conjunto de ícones minimalistas e consistentes.
* **Google Fonts (Inter)** - Tipografia focada em legibilidade.

## 📦 Como rodar o projeto

1.  Faça o download ou clone este repositório.
2.  Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estejam na mesma pasta.
3.  Abra o arquivo `index.html` em qualquer navegador moderno.
